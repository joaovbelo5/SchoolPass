Como Ativar os Avisos

Siga os passos abaixo para configurar o recebimento de mensagens. Este processo é feito apenas uma vez.

  1. Instale o Telegram no seu celular.
  2. Obtenha seu ID clicando aqui: @getmyidx_bot
    (Clique em "Iniciar" e copie o número recebido)
  3. Preencha o formulário abaixo com os dados da carteirinha e o ID copiado.
  4. Ative o bot de avisos: @GISNO_bot
    (Clique em "Iniciar" para autorizar o envio)

Pronto! Você receberá os avisos.
